# java-cli-millbuild-duckdb-simple

## Description
Reads from csv and json giving field names.

## Tech stack
- java
- millbuild
  - duckdb

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
